# 术语表

:::{glossary}

解释器
  执行 Python 源代码的程序（如 CPython）。

虚拟环境
  一个隔离的 Python 环境，用于为项目独立安装依赖。

变量
  程序运行过程中可以改变值的名字。

常量（约定）
  约定不应改变值的名字，常用全大写命名来提示。

赋值
  使用 `=` 把右侧对象绑定到左侧名字。

多重赋值
  一条赋值语句同时给多个变量绑定值，如 `a, b = 1, 2`。

交换赋值
  使用 `a, b = b, a` 交换两个变量的值。

标识符
  程序员为变量、函数、类等起的名字。

关键字
  Python 的保留字，不能作为标识符使用。

缩进
  通过空格表示代码层级，用来界定代码块。

代码块
  由相同缩进层级组成的一组语句，作为一个整体执行。

注释
  以 `#` 开头的说明文本，用于解释代码意图，不会被执行。

文档字符串（docstring）
  写在模块、类或函数开头的三引号字符串，用于说明用途。

字符串（str）
  由字符组成的不可变序列类型。

索引
  用位置编号访问序列元素的方式，从 `0` 开始计数。

切片
  用 `start:end:step` 取出序列的一段，得到新的序列。

列表（list）
  可变的有序容器，适合保存一组同类或相关数据。

元组（tuple）
  不可变的有序容器，适合保存固定记录。

字典（dict）
  键值对映射结构，通过键访问值，键必须唯一。

集合（set）
  不重复元素的无序容器，常用于去重与成员判断。

布尔值（bool）
  只有 `True` 与 `False` 两个值，用于表示真或假。

类型（type）
  对象所属的类别，决定它支持的操作与行为。

对象标识（id）
  `id(x)` 返回对象在运行时的身份标识，可用于理解“是否是同一个对象”。

条件表达式
  计算结果为 `True` 或 `False` 的表达式，常用于分支与循环。

分支（if/elif/else）
  根据条件选择不同的代码块执行。

比较运算符
  用于比较大小或相等性的一组运算符，如 `==`、`!=`、`<`、`>=`。

逻辑运算符
  用于组合条件的一组运算符，如 `and`、`or`、`not`。

短路求值
  `and` 与 `or` 在结果已确定时不会继续计算后续表达式。

循环（for/while）
  让代码块重复执行的一种控制结构。

可迭代对象
  能被 `for` 逐个取出元素的对象，如字符串、列表、字典与集合等。

range
  生成整数序列的可迭代对象，常用于控制循环次数。

break
  立即退出当前循环。

continue
  跳过本次循环的剩余语句，进入下一次循环。

函数（function）
  可复用的代码单元，可接收参数并返回结果。

参数
  调用函数时传入的数据，用于影响函数内部计算。

默认参数
  在函数定义时为参数提供默认值，调用时可省略该参数。

返回值
  函数通过 `return` 交给调用者的结果。

作用域
  名字在代码中可见与可访问的范围，如局部作用域与全局作用域。

模块（module）
  一个 Python 文件，通常以 `.py` 结尾，用于组织代码。

包（package）
  用目录组织的一组模块，用于更大规模的代码分层与复用。

import
  把模块或对象导入当前命名空间的语句。

主入口（__main__）
  `if __name__ == "__main__":` 常用于区分“被导入”与“直接运行”。

标准库
  Python 自带的常用模块集合，如 `os`、`math`、`json` 等。

可变类型
  内容可原地修改的类型，如 `list`、`dict`、`set`。

不可变类型
  内容不可原地修改的类型，如 `int`、`str`、`tuple`。

引用
  多个变量指向同一个对象时，对象的修改会同时影响这些变量。

浅拷贝
  只复制最外层容器结构，内部元素仍可能与原对象共享引用。

深拷贝
  递归复制内部对象，使新对象与原对象完全独立。

成员运算符（in / not in）
  判断元素是否属于容器，或判断键是否存在于字典。

排序
  按规则对序列元素排列，常用 `sort()` 或 `sorted()`。

推导式
  用紧凑语法生成容器的写法，如列表解析、集合解析与字典解析。

zip
  把多个可迭代对象并行对齐迭代，常用于配对或合并多列数据。

map
  对可迭代对象逐项应用函数，得到新的迭代结果。

filter
  按条件筛选可迭代对象的元素，得到新的迭代结果。

路径
  文件或目录在磁盘中的位置表示。

相对路径
  相对于当前工作目录的路径。

绝对路径
  从磁盘根目录开始的完整路径。

工作目录
  程序默认解析相对路径的目录。

文件句柄（file object）
  `open()` 返回的文件对象，用于读写文件内容。

上下文管理器（with）
  使用 `with` 管理资源生命周期，退出代码块时自动关闭资源。

文件模式
  打开文件的方式，如 `r`、`w`、`a`、`rb`、`wb`。

编码（encoding）
  字符与字节之间的转换规则，常用 `utf-8`。

文本文件
  以字符方式读写的文件，通常需要指定编码。

二进制文件
  以 `bytes` 方式读写的文件，不经过字符编码转换。

文件指针
  文件当前读写位置，可用 `tell()` 查看、用 `seek()` 移动。

目录遍历
  递归列出目录树内容的过程，常用 `os.walk()`。

类（class）
  描述对象结构与行为的模板。

对象（object）
  类的一个实例，包含数据（属性）与行为（方法）。

实例化
  通过类创建对象的过程。

构造函数（__init__）
  在创建对象时自动调用的方法，用于初始化实例属性。

实例属性
  属于对象自身的属性，每个对象各有一份。

类属性
  属于类本身的属性，由所有对象共享。

实例方法
  第一个参数为 `self` 的方法，用于操作实例状态。

类方法（@classmethod）
  第一个参数为 `cls` 的方法，常用于读写类属性或提供替代构造方式。

静态方法（@staticmethod）
  不接收 `self/cls` 的方法，适合放置与类相关的工具函数。

继承
  子类复用父类成员并扩展能力的机制。

方法重写（override）
  子类用同名方法替换父类实现，以改变行为。

super
  用于在子类中调用父类实现的辅助机制。

封装
  隐藏内部实现细节，只暴露必要接口以减少误用。

保护成员
  以 `_name` 命名的成员，约定仅供类内部或子类使用。

私有成员
  以 `__name` 命名的成员，会触发名称重整以增强封装。

名称重整（name mangling）
  `__name` 会被改写为 `_类名__name` 的机制，用于减少外部误访问。

MRO（方法解析顺序）
  多继承时 Python 查找方法的顺序规则。
:::
